# Offensive-Security

This repository focuses on **offensive security techniques** used to understand how cyber attacks are conducted in real-world environments.

The primary goal is **education and defensive awareness** — helping students analyze attacker behavior, recognize common attack patterns, and better understand system weaknesses.

## Hands-on Practice Guide

Practical labs and Confidence & Security of Information Systems exercises are available at:  
[https://github.com/dzokha/Documents/tree/main/SysAdmin](https://github.com/dzokha/Documents/tree/main/InfoSec)

## Disclaimer

⚠️ **For educational use only.**  
This repository does not promote or support illegal or unethical activities.  
All techniques are intended to improve understanding of cybersecurity defenses.
